# Cisco Modeling Labs OVA Installation

This guide covers importing the Cisco Modeling Labs (CML) OVA into VMware Workstation.

---

# Download Cisco Modeling Labs

Download Cisco Modeling Labs from Cisco:

[Cisco Modeling Labs Free Download](https://software.cisco.com/download/home/286193282/type/286326381/release/CML-Free)

The download contains the CML virtual appliance:

```
Cisco Modeling Labs.ova
```

The OVA already contains:

- Virtual machine configuration
- Virtual disk
- Operating system
- CML software

---

# Import the OVA

Open VMware Workstation:

```
File
> Open
```

Select:

```
Cisco Modeling Labs.ova
```

VMware will detect the appliance configuration.

---

# Deploy Virtual Appliance

Choose:

```
Import
```

Set the virtual machine name:

```
Cisco Modeling Labs
```

Select the storage location.

Recommended:

```
SSD storage
```

Allow VMware to complete the OVA extraction and deployment.
